# Innovate_KDSH_2026 - Narrative Consistency Analysis

## Quick Start
pip install -r requirements.txt
python main.py --test data/test.csv --output output/results.csv

## Project Structure
- main.py: Entry point
- src/: Source code modules
- data/: Input data
- output/: Generated results
- docs/: Documentation

## Requirements
Python 3.9+, Pathway 0.6.0+
